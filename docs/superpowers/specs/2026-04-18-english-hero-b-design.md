# English Hero B Design

**Goal:** 仅改英文版首页首屏，让其在不改变中文页面和站点主体结构的前提下，获得更强的 editorial 视觉层级与倾斜图片展示效果。

**Scope:**
- 只影响英文版本 Hero 区域。
- 中文版本保持现状。
- 不引入参考图中的背景纹理或整页背景改造。
- About / Experience / Projects / Research / Education 的主体结构保持不变。

## Visual Direction

采用用户在预览中选中的 `B` 方案：
- 左侧文字改为更强的首屏层级：小型引导语、超大英文名、紧凑的角色说明、简短补充介绍、双 CTA。
- 右侧头像改为倾斜海报卡片风格，强化封面感，但不做花哨背景。
- 整体仍使用当前网站的清爽浅色基底，只在首屏加入更明显的留白、字号比例和卡片阴影。

## Layout

### Left Column
- 保留英文人物信息作为核心内容。
- 名字在英文模式下拆成更强的视觉主标题。
- 角色说明与简介控制在首屏内，不做过长段落。
- CTA 使用两个动作：一个主按钮、一个次按钮。

### Right Column
- 使用当前头像资源，不新增图片来源。
- 图片外层改为带圆角、白边、阴影的“海报卡片”。
- 卡片默认轻微旋转，hover 时可有轻微回正或浮起效果。

## Interaction

- 首屏文字与图片都加入轻量入场动画。
- 倾斜图片卡片在 hover 时增强阴影与位移。
- 动效必须克制，不影响阅读和简历型网站的可信度。

## Implementation Notes

- 语言判断继续基于现有 i18n 逻辑，只在英文模式应用新版 Hero 排版。
- Hero 组件内部按语言分支渲染：中文走现有版本，英文走新版布局。
- 样式尽量收敛在 Hero 相关文件和全局样式少量补充中，避免影响其他 section。
- 不修改中文翻译文案。

## Validation

- 本地英文模式下首屏应出现新版大标题与倾斜图片。
- 中文模式下首屏保持现有布局不变。
- 构建通过，线上发布后英文页面可正常显示，无首屏白屏或布局错乱。
